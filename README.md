Hướng dẫn cài đặt
Tải project về máy và mở bằng Visual Studio.

Mở SQL Server Management Studio (SSMS), mở file Database_INVARIX.sql và bấm Execute để tạo Database.

Mở file CauHinhKetNoi.cs trong Visual Studio, sửa phần Server=(localdb)\MSSQLLocalDB thành tên Server SQL trên máy của bạn.

Bấm F5 để chạy chương trình. (Tài khoản mẫu: admin / Mật khẩu: 123).
